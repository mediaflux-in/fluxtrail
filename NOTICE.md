# NOTICE

## Credits
Capsule Context CLI (Capsule Nexus) includes integration with the **Graphify** knowledge graph tool.

## Attribution
- **Graphify:** Originally developed by Safi Shamsi (https://github.com/safishamsi/graphify).
- **Integration:** This tool reads and compacts the outputs generated by Graphify (e.g., `GRAPH_REPORT.md`) to provide architectural context to AI assistants.

## Code Usage
This project does not contain copied or derived source code from the Graphify repository. All interactions are performed via shell execution of the `graphify` CLI or direct reading of the generated report files.
